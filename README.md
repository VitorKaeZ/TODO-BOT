# TODOBOT

TODOBOT are a to-do-list and suggestion bot.

## DEPENDENCES

```
discord.js
dotenv
express
lowdb
```

## INSTALL

Install Nodejs and write the command in the console

```
npm install
```

Create an archive .env and set this

```
TOKEN_APP=
CLIENT_ID=
NOT_APPROVED_CHANNEL=
APPROVED_CHANNEL=
COMPLETED_CHANNEL=
INITIAL_CHANNEL=
```
